Detailed Description

Structure containing dimension sizes.

  • 4D objects (Nx, Ny, Nz, Nt).
  • 3D objects (Nx, Ny, Nz, 0) - Nt must be 0. If we don't sample in time we make 3D datasets, otherwise 4D.
  • 2D objects (Nx, Ny, 1, 0) - Nt must be 1 and Nz must be 1 since all datasets are stored internally as 3D.

The structure contains only POD, so no C++ stuff is necessary.
